# Changelog

## [1.0.0] - 2026-08-XX (planned)
### Added
- Initial unified platform release
- PEN-COMPARE certification framework (migrated from pen-compare v0.1.0)
- PEN-DISCOVER: IS110 sequence prediction via ESM-2 650M embeddings
- PEN-TARGET: hg38 bRNA target site scanner with gnomAD v4 safety check
- PEN-MONITOR: Europe PMC + LLM living database engine
- PEN-SAFE: COSMIC CGC + DepMap + BepiPred-3.0 safety analysis
- PEN-DESIGN: IS621/ISCro4 bRNA designer (verified scaffold from PDB 8WT6)
- PEN-DELIVER: evidence-RAG delivery recommender
- PEN-BENCH: experimental protocol generator
- Unified Pipeline.run() end-to-end query interface
- FastAPI REST gateway
- Streamlit 10-tab webserver
- Community evidence portal

## [pen-compare 0.1.0] - 2026-06-XX

- ISCro4 = sole TRUE_WRITER; robustness = 1.000
