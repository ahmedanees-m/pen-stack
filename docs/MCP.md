# PEN-STACK MCP server

PEN-STACK exposes its validated tools over the **Model Context Protocol** so *any* external agent
(Claude, etc.) can plan genome writes against the same validated models — turning the platform into
shared infrastructure for the agentic ecosystem.

## Run
```bash
pip install 'pen-stack[services]'      # fastmcp
python -m pen_stack.agent.mcp_server   # or: docker compose up mcp
```

## Tools exposed

| Tool | Signature | Returns |
|---|---|---|
| `writability` | `(gene, ct="k562")` | most-writable locus near a gene (safety × durability) |
| `reachable_writers` | `(gene, ct="k562")` | writer families that can reach the gene's best locus |
| `writer_axes` | `(family)` | measured axes for a writer family (cargo, deliverability, tier) |
| `plan_write` | `(gene, intent, cargo_bp=2000, ct="k562")` | full Write Planner: ranked, traceable plan |
| `ask_literature` | `(q)` | grounded, cited literature answer (numbers still from tools) |

Every result is JSON and carries a `tool` provenance tag. Numbers are produced by the validated
modules — never generated by a language model. Clients should surface the decision-support disclaimer.
